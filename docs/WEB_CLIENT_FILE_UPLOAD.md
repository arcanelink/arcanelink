# Web 客户端文件发送功能使用说明

## 功能概述

Web 客户端现已支持在聊天界面直接发送文件，包括：
- 📎 通用文件
- 🖼️ 图片（自动预览）
- 🎵 音频（内置播放器）
- 🎬 视频（内置播放器）

## 使用方法

### 1. 发送文件

在聊天窗口底部的输入框左侧，点击 📎 按钮：

```
[📎] [😀] [输入框...] [Send]
```

1. 点击 📎 按钮
2. 选择要发送的文件（最大 50MB）
3. 等待上传完成
4. 文件会自动发送到当前聊天

### 2. 查看文件消息

#### 图片消息
- 自动显示图片预览
- 点击图片可查看原图
- 显示文件名和大小

#### 音频消息
- 显示音频播放器
- 可直接在聊天中播放
- 显示文件名和大小

#### 视频消息
- 显示视频播放器
- 可直接在聊天中播放
- 显示文件名和大小

#### 通用文件
- 显示文件图标和信息
- 点击下载按钮下载文件
- 显示文件名和大小

### 3. 下载文件

对于通用文件消息，点击消息中的 ⬇️ 按钮即可下载。

## 支持的文件类型

系统会根据文件的 MIME 类型自动识别：

- **图片**: image/* (jpg, png, gif, webp 等)
- **音频**: audio/* (mp3, wav, ogg 等)
- **视频**: video/* (mp4, webm, mov 等)
- **其他**: 作为通用文件处理

## 文件大小限制

- 单个文件最大 50MB
- 超过限制会提示错误

## 技术实现

### 前端组件

1. **ChatWindow.tsx**
   - 添加文件上传按钮
   - 处理文件选择和上传
   - 自动识别文件类型
   - 发送文件消息

2. **MessageItem.tsx**
   - 渲染不同类型的文件消息
   - 图片预览
   - 音频/视频播放器
   - 文件下载链接

3. **API Client**
   - `uploadFile()` - 上传文件
   - `getFileInfo()` - 获取文件信息
   - `getFileDownloadUrl()` - 获取下载链接

### 消息格式

#### 图片消息
```json
{
  "msgtype": "m.image",
  "body": "photo.jpg",
  "url": "/_api/v1/files/FILE_ID",
  "info": {
    "size": 1024000,
    "mimetype": "image/jpeg"
  }
}
```

#### 音频消息
```json
{
  "msgtype": "m.audio",
  "body": "song.mp3",
  "url": "/_api/v1/files/FILE_ID",
  "info": {
    "size": 5120000,
    "mimetype": "audio/mpeg"
  }
}
```

#### 视频消息
```json
{
  "msgtype": "m.video",
  "body": "video.mp4",
  "url": "/_api/v1/files/FILE_ID",
  "info": {
    "size": 10240000,
    "mimetype": "video/mp4"
  }
}
```

#### 通用文件
```json
{
  "msgtype": "m.file",
  "body": "document.pdf",
  "url": "/_api/v1/files/FILE_ID",
  "info": {
    "size": 2048000,
    "mimetype": "application/pdf"
  }
}
```

## 样式说明

文件消息使用特殊的样式：

- **文件卡片**: 圆角背景，带图标和信息
- **图片**: 最大宽度 100%，最大高度 300px
- **音频**: 宽度 300px 的播放器
- **视频**: 最大宽度 100%，最大高度 300px
- **下载按钮**: 悬停时高亮显示

## 注意事项

1. **文件大小**: 确保文件不超过 50MB
2. **网络状态**: 上传大文件时请保持网络连接
3. **浏览器兼容**: 建议使用现代浏览器（Chrome, Firefox, Safari, Edge）
4. **文件格式**: 某些特殊格式可能无法预览，但可以下载

## 错误处理

- 文件过大：显示 "File size exceeds 50MB limit"
- 上传失败：显示 "Failed to upload file" 及错误信息
- 网络错误：自动重试或提示用户
